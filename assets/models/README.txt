# IMPORTANTE: Inserisci il file environment.glb in questa cartella

Il modello 3D deve:
- Essere in formato .glb o .gltf
- Contenere un oggetto chiamato "monitor" o "screen"
- Essere ottimizzato (< 10 MB possibilmente)
- Avere le texture embed (per .glb) o nella cartella textures (per .gltf)

Se non hai ancora il modello, puoi:
1. Crearlo in Blender
2. Scaricarne uno da Sketchfab o similari
3. Usare un modello placeholder per testare
